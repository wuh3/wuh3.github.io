Remote Controlled Kart with Object Bypass System

As the final project of my Mechatronics class, I built a remote controlled kart using 3D printing and Arduino AtMega 2560. The chassis is totally designed by myself using SolidWorks. I 3D printed it and assembled all the parts. The kart is powered by two DC motors. The steering is controlled by a servo motor. I implemented the IR Remote library in Arduino to allow the kart to be controlled by a remote. A supersonic sensor was used to detect objects in front of the kart. Then, system will implement algorithms to bypass the object by calculating the distance and speed of the kart.
